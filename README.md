### Программа для поиска модильных номеров в текстовом файле

Паттерн поиска определяется регулярным выражением, `[78]?\d{3}\D?\d{3}\D?\d{2}\D?\d{2}`, который позволяет искать в текстовом файле следущие форматы записей номеров телефона:

```
Слитный            9963060722
С разделителями    996?306?07?22
С пробелами        996 306 07 22
С тире             996-306-07-22
Со скобками        (996)306-07-22
С кодом страны     +7(996)306-07-22
                   8(996)306-07-22
Смешанные          996.306.07.22
```

### Сборка программы для использования в ОС GNU/Linux (x86_64)
```bash
# Базовая сборка в исполняемый файл
GOOS=linux GOARCH=amd64 go build -o gotelsfromfile .

# Сборка без использования зависимостей при исполнении
CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build -o gotelsfromfile .

# Сборка с уменьшением объема исполняемого файла
# флаг "-s" - убирает символы отладки (уменьшает размер)
# флаг "-w" - убирает DWARF-информацию (ещё сильнее уменьшает размер)
CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build \
    -ldflags="-s -w" \
    -o gotelsfromfile \
    .

# При необходимости дальнейшего уменьшения размера исполняемого файла,
#   можно применить компрессор upx, после его предварительной устанвки в систему
#   sudo apt-get install upx -y
upx --brute myapp-linux-amd64
```

### Сборка программы для использования в ОС Windows (x86_64)

```bash
# Статическая сборка
# Важно: для Windows CGO_ENABLED=0 особенно полезен,
# так как отключает зависимость от Visual C++ # Redistributable (MSVCRT).
CGO_ENABLED=0 GOOS=windows GOARCH=amd64 go build -o gotelsfromfile.exe .
```

#### Сборка для разных архитектур Windows
```bash
# для архитектуры 32-bit (x86)
GOOS=windows GOARCH=386 go build -o gotelsfromfile.exe .

# для архитектуры 64-bit (amd64)
GOOS=windows GOARCH=amd64 go build -o gotelsfromfile.exe .
```
